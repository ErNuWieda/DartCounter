# Projekt-Roadmap für Dartcounter

Dieses Dokument skizziert geplante Features und Verbesserungen für zukünftige Versionen des Dartcounters.

---

## Version 1.2.0 - "The Foundation" (Veröffentlicht)

Dieses Release hat das Fundament für alle zukünftigen Entwicklungen gelegt. Der Fokus lag auf einer robusten Architektur, einer breiten Palette an Spielmodi und einer professionellen Code-Qualität.

- **Implementiert:** Umfassende Spielmodi (X01, Cricket, Killer, etc.), Speichern/Laden, Spielerstatistiken, Highscores, Theming.
- **Technisch:** MVC-Architektur, Test-Suite (>115 Tests), Build-System, umfassende Dokumentation.
---

## Version 1.3.0 - "The Tournament Edition" (Geplant)

Der Fokus für v1.3.0 liegt auf der Erweiterung der sozialen und kompetitiven Aspekte des Spiels.

### ✨ Geplante Haupt-Features

- **[ ] Turnier-Modus:**
  - Implementierung eines einfachen K.O.-Turnierbaums für 4, 8 oder 16 Spieler.
  - Automatische Generierung von Spielpaarungen.

- **[ ] Spieler-Profile & Avatare:**
  - Möglichkeit, Spieler-Profile zu speichern (nicht nur Namen).
  - Jedem Spieler einen Avatar aus einer vordefinierten Auswahl zuweisen.

- **[ ] Erweiterte Statistiken:**
  - Eine "Heatmap" für das Dartboard, die anzeigt, welche Felder ein Spieler am häufigsten trifft.
  - Detailliertere Checkout-Weg-Vorschläge für X01-Spiele.
